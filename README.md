# (C# 코딩) 
	Login Screen
## 개요
	-C# 프로그래밍학습
	-1줄소개: 사용자에게 아이디와 비밀번호를 입력받아 로그인 기능을 구현하는 프로그램입니다.

	-사용한컨트롤:-Label, TextBox, ListBox, Button
